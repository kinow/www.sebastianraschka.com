Automated Desktop Spring Cleaning

-- written by Sebastian Raschka on April 6, 2013

Tweet

When you desktop became so cluttered that you can barely see the wallpaper behind the bulk of different files, it is about time for a little bit of spring cleaning.



I find it very convenient to have a script on my desktop - just one double-click away - that does this job for me. If executed, it will first create 3 folders on the Desktop: PDF Documents, Pictures, and Word Documents (if not already available). Then, it will move all picture files into Pictures, all Word documents into Word Documents, and all PDFs into PDF Documents, respectively. Simple and easy, but very efficient!


You can download the script from HERE, or you can create it yourself:
Simply copy the following lines into you favorite text editor and save it as cleanDesktop.command



cd ~/Desktop
mkdir Pictures
mkdir PDF\ Documents
mkdir Word\ Documents

mv *.png *.bmb *.jpeg *.jpg *.tiff *.tiff Pictures
mv *.pdf PDF\ Documents
mv *.doc *.docx Word\ Documents
exit 0

Then open a bash shell (terminal) and make the script executable:
chmod ugo+x /location/to/the/file/cleanDesktop.command

- The script works on MacOS X and Linux operating systems.
