Text Cleaner

-- written by Sebastian Raschka on June 2, 2013

Tweet

Text Cleaner is a command line program that removes all unwanted symbols and non-ASCII characters from a text file.

This program is especially useful if you have a whole bunch of files that were created by a particular program and contain some weird symbols, characters, and blank lines that make those files incompatible with another, second program.
Because Text Cleaner is a command line tool, it can be easily included in other scripts, which can help you automating your work processes.
