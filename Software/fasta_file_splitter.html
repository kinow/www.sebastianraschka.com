FASTA File Splitter

-- written by Sebastian Raschka on July 22, 2013

Tweet

FASTA File Splitter is a command line is a command line program that let's you split a multi-entry FASTA file into individual files. within fractions of a second.

A folder with the results will be created in the same folder where your input FASTA file residues, and the files will be names after their FASTA sequences.
It is very easy to use, simply launch it from your terminal and follow the on-screen instructions.
