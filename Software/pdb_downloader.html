PDB Downloader

-- written by Sebastian Raschka on May 21, 2013

Tweet

The PDB Downloader lets you download multiple PDB files at once.

Here is how it works:

1) Prepare a list with the 4-letter PDB codes for the target protein structure files that you want to download.
The formatting does not matter as long as the file is saved as a plain text file.

2) Start the program and enter the location of your list file.

3) Choose whether you want to download the whole PDB file, only the protein (ATOM entries), or only the ligands (HETATM entries)

You will see a progress bar appearing at the bottom of your terminal that will give you some idea how long the download process might take. After all PDB files were downloaded, you will be notified where they were saved - the program creates a subfolder in you current working directory. Also, the program will notify you if some files could not been downloaded.
