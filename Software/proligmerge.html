Protein-Ligand Merge

-- written by Sebastian Raschka on July 22, 2013

Tweet

ProLig Merge is a command line is a command line program that let's you merge protein and ligand protein structure files (PDBs).

This tool allows you to append ligands to protein files, and you don't have to worry about renumbering the atom numbers, ProLig Merge takes care of it automatically.



Because ProLig Merge is a command line tool, it can be easily included in other scripts, which can help you automating your work processes.
